# Variables, Constructors & Access Modifiers

### Variables
* **Local**: Variables declared inside a method.
* **Instance**: Variables declared inside a class but outside any method.
* **Class**: Static variables declared within a class that belong to the class itself rather than any instance; they cannot be local.

### Constructor
* **Initialization**: Used to set initial values for an object's attributes.
* **Types**: Parameterized and Default.

### Access Modifiers
Control the visibility and accessibility of classes, methods, variables, and constructors.
* **Types include**: `public`, `protected`, `default`, and `private`.
