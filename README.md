# Placement Training

This repository contains standard Java programs implemented for technical interview preparation, focusing on key computational algorithms, base conversions, primality tests, and number theory exercises.

## Repository Structure

- **Conversions/**: Implementations for base conversion (Binary and Decimal).
- **Primes/**: Various algorithms for primality checks, cyclic primes, and list generation.
- **SpecialNumbers/**: Implementations of mathematical properties including Armstrong, Automorphic, Harshad, and Perfect numbers.

## Compilation and Execution

Compile and run any Java file using the Java Development Kit (JDK):

```bash
javac SpecialNumbers/PerfectNumber.java
java SpecialNumbers.PerfectNumber
```
